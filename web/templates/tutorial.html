Руководство по игре в OFC Poker

Основные правила

Open Face Chinese Poker (OFC) - это покерная игра, где каждый игрок должен составить три покерных комбинации, располагая карты в три ряда:

  • Верхний ряд (3 карты)
  • Средний ряд (5 карт)
  • Нижний ряд (5 карт)

Главное правило: сила комбинаций должна увеличиваться снизу вверх. То есть нижний ряд должен быть сильнее среднего, а средний сильнее верхнего.

Как играть

  1. Выберите режим игры и количество игроков
  2. Настройте параметры AI и время на ход
  3. Разместите карты в ряды, соблюдая правило старшинства
  4. Следите за статистикой и анализируйте свою игру

Фантазия

Фантазия - особый режим игры, который активируется при определенных условиях:

  • Нормальная фантазия: активируется при сборе сильных комбинаций
  • Прогрессивная фантазия: даёт дополнительные бонусы за последовательные успешные фантазии

В режиме фантазии игрок получает дополнительные карты для размещения.

Статистика

В игре отслеживаются различные показатели:

  • Fouls Rate - процент ходов, нарушающих правило старшинства комбинаций
  • Scoops Rate - процент ситуаций, когда игрок выигрывает все три ряда
  • Win Rate - процент выигранных игр
  • Fantasy Rate - процент успешных фантазий

Управление

Мышь/Тачскрин:

  • Перетащите карту в нужную позицию
  • Кликните по карте для выбора
  • Кликните по слоту для размещения выбранной карты

Клавиатура:

  • A-K-Q-J-T-9-8-7-6-5-4-3-2 - выбор ранга карты
  • H-S-D-C - выбор масти
  • Ctrl+Z - отменить последний ход
  • Esc - отменить выбор
  • Enter - подтвердить ход

Настройки

Основные параметры:

  • Animation:
    • Normal - обычная скорость анимации
    • Fast - ускоренная анимация
    • Off - анимация отключена
  • AI Think Time - время на ход AI
  • Fantasy Type - тип фантазии (Normal/Progressive/Disabled)
  • Sound Effects - включение/выключение звуков

Советы для начинающих

  • Начинайте с размещения сильных комбинаций в нижнем ряду
  • Следите за вышедшими картами
  • Используйте режим тренировки для практики
  • Анализируйте статистику для улучшения игры
  • Старайтесь минимизировать количество фолов
  • Ищите возможности для скупов

Мобильная версия

На мобильных устройствах доступны дополнительные жесты:

  • Свайп влево - открыть боковую панель
  • Свайп вправо - закрыть боковую панель
  • Долгое нажатие на карту - показать возможные ходы
  • Двойное нажатие - отменить выбор
